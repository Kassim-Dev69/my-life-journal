# 🤝 Comment contribuer à ce journal

Ce journal est personnel, mais ouvert aux retours et suggestions !

- 💡 **Proposer une idée** : ouvre une [Discussion](https://github.com/Kassim-Dev69/my-life-journal/discussions)
- 🐛 **Signaler une coquille** : ouvre une Issue avec le tag `typo`
- ✍️ **Partager une ressource** : commente une entrée ou envoie un PR dans `resources/`

Merci de faire partie du voyage ! 🚀
